# MacCam

Нативная macOS security-камера на базе веб-камеры Mac. Живёт в меню-баре,
детектирует движение и пишет видеоклипы на диск. Полностью офлайн — никаких
облаков, аккаунтов и сети.

## Возможности

- Захват с **максимального доступного разрешения** камеры (опрашивает устройство,
  не хардкодит 1080p — внешние USB/Continuity камеры до 4K поддерживаются автоматически).
- Детекция движения на vImage (даунскейл 320×180 + покадровая разность), throttle ~12 Гц.
- Запись клипов по движению через `AVAssetWriter`, кодек **HEVC** (аппаратный энкодер M1)
  или H.264. Ротация по максимальной длине, cooldown по «тишине», опциональный pre-roll.
- Меню-бар с индикацией состояния (серый / зелёный / мигающий красный при записи).
- Режим охраны: авто-старт при блокировке экрана.
- Запуск при логине (`SMAppService`).
- App Sandbox, без сетевых entitlements.

## Сборка

```sh
xcodebuild -project MacCam.xcodeproj -scheme MacCam -configuration Debug \
  -destination 'platform=macOS' build
```

Или открыть `MacCam.xcodeproj` в Xcode и запустить (⌘R). Требуется macOS 13.0+.

## Тесты

```sh
xcodebuild -project MacCam.xcodeproj -scheme MacCam -destination 'platform=macOS' test
```

Покрыта чистая логика: маппинг чувствительности, битрейт-пресеты, выбор max-формата,
кольцевой буфер, имена/очистка клипов, конечный автомат записи и детектор движения
на синтетических кадрах.

## Использование

1. При первом запуске разрешите доступ к камере (и микрофону, если включён звук).
2. Иконка появляется в меню-баре. **Start Monitoring** — начать наблюдение.
3. При движении в кадре пишется клип в папку (по умолчанию `~/Movies/MacCam/`).
4. **Settings…** — камера, чувствительность, длины клипов, cooldown, pre-roll,
   звук, FPS, кодек/качество, папка, автоочистка, режим охраны, запуск при логине.
5. **Open Clips Folder…** — открыть папку с записями.

## Приватность

Весь код офлайн: нет сетевых entitlements, нет загрузок/синхронизаций. Файлы хранятся
только локально в выбранной папке (поддержка внешних дисков/NAS через security-scoped
bookmark).
